# rock-paper-scissor
A responsive rock paper scissor game built using HTML, CSS, and JavaScript.

## Live Demo
https://somyaa1207.github.io/rock-paper-scissor/

## Tech Stack
- HTML
- CSS
- JavaScript

##  Features
- Interactive gameplay
- User vs Computer logic
- Score tracking
- Responsive design

##  How to Play
1. Choose Rock, Paper, or Scissors  
2. Computer will make a choice  
3. Result will be displayed instantly
